Hash in url?

It works pretty simple. Each map is generated by some rules that change randomness. Those rules are showed in url after # and you read them like this: #WIDTH|HEIGHT|SEED|ISLAND|ISLANDS_SIZE|GRASS|PALMS|CHESTS.

  • WIDTH & HEIGHT - size of map (in blocks)
  • SEED - whatever string for random generator
  • ISLANDS - number of points to start generating terrain
  • ISLANDS_SIZE - generator expand each "island" using this number
  • GRASS, PALMS - generator use this numbers to push/reduce randomness (there's more rules for grass and palms)
  • CHESTS - actual number of chests (if there is space for them)

You can change those numbers and experiment with results or simply press the [random] button in menu ( recommended ). Some values may don't work while some may results in awesome maps.
